# Pothole Detector CV

This project detects potholes in road images using YOLOv8.

## 🚀 Features
- Object detection using YOLOv8
- Trained on Roboflow dataset
- Real-time inference with Gradio UI

## 📂 Project Structure
- `data/` - Dataset
- `src/` - Source code
- `notebooks/` - Experiments
- `results/` - Outputs
